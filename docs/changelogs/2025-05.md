### May 2, 2025 

**What's New:**

* **Webflow Publishing Enhancements:** We've expanded our Webflow integration to support multiple collection selection. You can now publish documentation updates to several Webflow collections simultaneously, making it easier to maintain consistent documentation across different sections of your Webflow site.

**Improvements:**

* **Webflow Collection Management:** The Webflow integration now provides a more intuitive interface for selecting and managing collections, with support for multi-collection publishing from a single project.
