# Zendesk Integration

Promptless integrates with Zendesk through OAuth 2.0, enabling automated documentation updates based on support ticket patterns and customer interactions.

## Installation

1. Click "Connect Zendesk" from the [integrations page](https://app.gopromptless.ai/integrations).
<img src="/img/zendesk-integrations-blank.png" alt="Integrations Page" style={{width:"800px", display: "block", margin: "0 auto", boxShadow: "0 4px 8px rgba(0, 0, 0, 0.1)", marginBottom: "20px"}} />

2. Enter your Zendesk subdomain and click continue.
{/* <img src="/img/zendesk-subdomain.png" alt="Zendesk Subdomain Entry" style={{width:"800px", display: "block", margin: "0 auto", boxShadow: "0 4px 8px rgba(0, 0, 0, 0.1)", marginBottom: "20px"}} /> */}

3. Review and approve the requested permissions on the Zendesk authorization page.

4. Verify that Zendesk is connected in the integrations page.

## Project Setup and Permissions

After connecting Zendesk, you can configure ticket monitoring settings in the [projects page](https://app.gopromptless.ai/projects). When setting up a project, you can:

- Select which ticket fields to monitor
- Configure ticket tags that trigger documentation updates
- Set up custom field mappings
- Define documentation update rules based on ticket patterns

## Authentication Model

Promptless uses Zendesk's [official OAuth 2.0 authentication](https://developer.zendesk.com/api-reference/ticketing/introduction/#security-and-authentication), which provides:

- Secure token-based authentication
- Granular API access scopes
- Ability to revoke access at any time
- Regular token rotation for enhanced security
