# Document360 Integration

**Used for: Publishing**

Promptless integrates with Document360 through API token authentication, enabling automated publishing of documentation updates to your Document360 knowledge base.

## Installation

1. Click "Connect Document360" from the [integrations page](https://app.gopromptless.ai/integrations).
2. In the connection modal, you'll need to provide your Document360 API token.
3. To generate an API token:
   - Log into your Document360 account
   - Follow the instructions in the [Document360 API documentation](https://apidocs.document360.com/apidocs/api-token)
   - Copy your generated API token
4. Paste the API token into the connection modal and click "Connect"
5. Verify that Document360 is connected in the integrations page

## Project Setup

After connecting Document360, you can configure publishing settings in the [projects page](https://app.gopromptless.ai/projects). When setting up a project, you can configure Promptless to automatically publish documentation updates to your Document360 knowledge base.

## Authentication Model

Promptless uses Document360's API token-based authentication, which provides:

- Secure API access
- Ability to revoke access by regenerating the token
- Granular permission controls through Document360's token management
