# Linear Integration

Promptless integrates with Linear to provide context-aware documentation updates based on your project management data.

## Authentication

- Uses OAuth 2.0 for secure authorization
- Read-only access to issue data
- No permanent storage of issue content

## Features

- **Issue Context**: Use Linear issues for documentation context
- **Project Alignment**: Keep documentation in sync with project status
- **Automated References**: Link documentation to relevant issues

## Security Features

- **OAuth 2.0**: Secure authorization protocol
- **Read-only Access**: Limited to viewing issue data
- **Data Processing**: No permanent storage of issue content
- **Encryption**: All data encrypted in transit and at rest

## Configuration

1. Connect your Linear account through OAuth
2. Select which teams/projects to monitor
3. Configure documentation update triggers
4. Set up notification preferences

## Best Practices

- Regular review of access permissions
- Monitor Linear audit logs
- Configure appropriate issue visibility settings

For additional security information, refer to our [Security and Privacy documentation](/security-and-privacy).
